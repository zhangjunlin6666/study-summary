
# 表格

注： : 代表对齐方式 ,: 与 | 之间不要有空格，否则对齐会有些不兼容。

1、复杂写法

代码：

    |a|b|c|
    |:------:|:------|------:|
    |居中|居左|居右|
    |======|======|======|

演示：

|a|b|c|
|:------:|:------|------:|
|居中|居左|居右|
|======|======|======|

2、简约写法

代码：

    a|b|c
    :-:|:-|-:
    居中|居左|居右
    ======|======|======

演示：

a|b|c
:-:|:-|-:
居中|居左|居右
======|======|======

3、特殊表格
