
# 行内标记

如果要标记一小段行内代码（显示在同一行），可以用反引号把它抱起来(`)，例如：

代码：

    单行：Use the `printf()` function.

演示：

Use the `printf()` function.

代码：

    多行：
        `
        < div>
            < div></div>
            < div></div>
            < div></div>
        < /div>
        `
演示：

`
< div>
    < div></div>
    < div></div>
    < div></div>
< /div>
`
